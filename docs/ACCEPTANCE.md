## M3 发布、搜索、子指南与学习模式

- [ ] 草稿发布生成不可变递增版本；搜索只返回已发布指南。

- [ ] 展开无 ID 冲突、坐标正确、入口接线和来源追踪；再次展开幂等；折叠只隐藏展开产物。
- [ ] 上游再发布不会改变已有引用，且测试覆盖。
- [ ] 学习者可搜索进入并按步骤浏览，点击视频关键点会跳转。
